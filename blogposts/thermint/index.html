Thermodynamic Integration

Thermodynamic Integration (TI) is a method coming straight from statistical physics to compute the difference of free energy between two systems AAA and BBB. We are going to go from this physics definition and see how we can apply it to Bayesian Inference.

We start with a system with a potential energy function U(x)U(x)U(x) with associated probability function

p(x)=1Zexp⁡(−U(x)kBT)p(x) = \frac{1}{Z}\exp\left(-\frac{U(x)}{k_BT}\right)p(x)=Z1​exp(−kB​TU(x)​)

Where ZZZ is the partition function of this system defined as

Z=∫Ωexp⁡(−U(x)kBT)dxZ = \int_\Omega \exp\left(-\frac{U(x)}{k_BT}\right)dxZ=∫Ω​exp(−kB​TU(x)​)dx

where Ω\OmegaΩ is the domain of all the states of the system and xxx is one state of this system. The definition of the free energy is given by

F=−kBTlog⁡(Z)F = -k_BT\log(Z)F=−kB​Tlog(Z)

Now going back to our systems AAA and BBB with potential energy UAU_AUA​ and UBU_BUB​ . We can construct a new system built as a linear interpolation between the two systems : Uλ=UA+λ(UB−UA){U_\lambda = U_A + \lambda(U_B - U_A)}Uλ​=UA​+λ(UB​−UA​)

We can show that

FB−FA=∫01EU(λ)[UB−UA]dλF_B - F_A = \int_0^1 E_{U(\lambda)}\left[U_B - U_A\right]d\lambdaFB​−FA​=∫01​EU(λ)​[UB​−UA​]dλ

It is all very nice, but very abstract if you are not a physicist! What does it have to do with Bayesian inference?

The Bayes theorem states that the posterior

p(x∣y)=p(x,y)p(y)p(x|y) = \frac{p(x,y)}{p(y)}p(x∣y)=p(y)p(x,y)​ ,

where xxx is my hypothesis or the parameters of my model, and yyy is my observed data! We can actually rewrite as an energy model, as earlier!

p(x∣y)=1Zexp⁡(−U(x,y))p(x|y) = \frac{1}{Z}\exp\left(-U(x,y)\right)p(x∣y)=Z1​exp(−U(x,y))

where the potential energy U(x,y)=−log⁡(p(x,y))U(x,y) = -\log(p(x,y))U(x,y)=−log(p(x,y)) is the negative log joint and the partition function Z=∫Ωexp⁡(−U(x,y))dxZ=\int_\Omega \exp(-U(x,y))dxZ=∫Ω​exp(−U(x,y))dx is the evidence. We have furthermore the connection that the free energy is nothing else than the log evidence.

Now we have a direct connection between energy models and Bayesian problems! Which leads us to the use of TI for Bayesian inference:

The most difficult part of the Bayes theorem comes from the fact that except for simple cases, the posterior p(x∣y)p(x|y)p(x∣y) is intractable. Most of the time the joint distribution p(x,y)p(x,y)p(x,y) is known in closed form, the real issue is then to estimate the evidence p(y)p(y)p(y), which is exactly what TI aims at.

The two systems we are going to consider is the prior : UA(x)=−log⁡p(x)U_A(x) = -\log p(x)UA​(x)=−logp(x) and the joint distribution UB(x)=−log⁡p(x,y)=−log⁡p(y∣x)−log⁡p(x)U_B(x) = -\log p(x,y) = -\log p(y|x) - \log p(x)UB​(x)=−logp(x,y)=−logp(y∣x)−logp(x). Now our intermediate state is given by

Uλ(x)=−log⁡p(x)+λ(−log⁡p(y∣x)−log⁡p(x)−log⁡p(x))=−log⁡p(x)−λp(y∣x)U_\lambda(x) = -\log p(x) + \lambda(-\log p(y|x) - \log p(x) - \log p(x)) = -\log p(x) - \lambda p(y|x)Uλ​(x)=−logp(x)+λ(−logp(y∣x)−logp(x)−logp(x))=−logp(x)−λp(y∣x).

The normalized density derived from this potential energy is called a power posterior :

pλ(x∣y)=p(y∣x)λp(y)Zλp_\lambda(x|y) = \frac{p(y|x)^\lambda p(y)}{Z_\lambda}pλ​(x∣y)=Zλ​p(y∣x)λp(y)​

which just a posterior for which we reduced the importance of the likelihood.

Ok so applying the thermodynamical integration we got earlier this gives us :

−log⁡p(y)+log⁡p(x)=∫01Epλ(x)[log⁡p(y∣x)]dλ - \log p(y) + \log p(x) = \int_0^1 E_{p_\lambda(x)}[\log p(y|x)]d\lambda−logp(y)+logp(x)=∫01​Epλ​(x)​[logp(y∣x)]dλ

Reformulated to get the log evidence :

log⁡p(y)=log⁡p(x)−∫01Epλ(x)[log⁡p(y∣x)]dλ \log p(y) = \log p(x) - \int_0^1 E_{p_\lambda(x)}[\log p(y|x)]d\lambda logp(y)=logp(x)−∫01​Epλ​(x)​[logp(y∣x)]dλ

Which is the log partition of the prior plus the continuous contribution to go from the prior to the posterior!

Now let's put this to practice for a very simple use case a Gaussian prior with a Gaussian likelihood

p(x)=N(x∣10,1)p(y∣x)=N(y∣x,1)\begin{aligned} p(x) = \mathcal{N}(x|10, 1) p(y|x) = \mathcal{N}(y|x, 1) \end{aligned}p(x)=N(x∣10,1)p(y∣x)=N(y∣x,1)​

We take y=−10y = -10y=−10 to make a clear difference between UAU_AUA​ and UBU_BUB​

Of course the posterior can be found analytically but this will help us to evaluate different approaches. 

using Plots, Distributions, LaTeXStrings; pyplot(); default(lw = 2.0, legendfontsize = 15.0, labelfontsize = 15.0)
σ_prior = 1.0; σ_likelihood = 1.0
μ_prior = 10.0
prior = Normal(μ_prior, σ_prior)
likelihood(x) = Normal(x, σ_likelihood)
y = -10.0
p_prior(x) = pdf(prior, x)
p_likelihood(x, y) = pdf(likelihood(x), y)

Now we have all the parameters in place we can define the exact posterior and power posterior:

σ_posterior(λ) = sqrt(inv(1/σ_prior^2 + λ / σ_likelihood^2))
μ_posterior(y, λ) = σ_posterior(λ) * (λ * y / σ_likelihood^2 + μ_prior / σ_prior^2)
posterior(y) = Normal(μ_posterior(y, 1.0), σ_posterior(1.0))
p_posterior(x, y) = pdf(posterior(y), x)
power_posterior(y, λ) = Normal(μ_posterior(y, λ), σ_posterior(λ))
p_pow_posterior(x, y, λ) = pdf(power_posterior(y, λ), x)
xgrid = range(-10, 10, length = 400)
plot(xgrid, p_prior, label = "Prior", xlabel = "x")
plot!(xgrid, x->p_likelihood(x, y),label =  "Likelihood")
plot!(xgrid, x->p_posterior(x, y), label = "Posterior")
plot(xgrid, [x->p_pow_posterior(x, y, λ) for λ in 0:0.2:1], label = reshape(["λ = $λ" for λ in 0:0.2:1], 1, :), title = "Power Posteriors", xlabel = "x", ylabel = L"p_\lambda(x|y)")

We can also visualize the energies themselves 

U_A(x) = -logpdf(prior, x)
U_B(x, y) = -logpdf(likelihood(x), y) - logpdf(prior, x)
U_λ(x, y, λ) = -logpdf(prior, x) - λ * logpdf(likelihood(x), y)
plot(xgrid, U_A, label = "U_A", xlabel = "x", ylabel = L"U(x)")
plot!(xgrid, x->U_B(x, y), label = L"U_B")  
plot!(xgrid, [x->U_λ(x, y, λ) for λ in 0:0.2:1], label = reshape(["λ = $λ" for λ in 0:0.2:1], 1, :), title = "Potential Energy")

Now we can start evaluating the integrand for multiple λ\lambdaλ :

λs = range(0, 1, length= 1000)
expec_λ = zeros(length(λs))
T = 1000
for (i, λ) in enumerate(λs)
    pow_post = power_posterior(y, λ)
    expec_λ[i] = sum(logpdf(likelihood(x), y) for x in rand(pow_post, T)) / T
end
plot(λs, expec_λ, label = L"E_{p_\lambda}[\log p(y|x)]", xlabel = L"\lambda")

And we can now compare the sum with the actual value of ZZZ :

using Trapz
logZ = logpdf(posterior(y), y)
TI_logZ = logpdf(prior, mean(prior)) .- [trapz(expec_λ[1:i], λs[1:i]) for i in 1:length(λs)]
plot(λs, TI_logZ, label = "Therm. Int.", xlabel = L"\lambda")
hline!([logZ], label = "logZ = $logZ")
